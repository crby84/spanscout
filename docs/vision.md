# SpanScout Vision

SpanScout ist ein **Observability Platform Prototype**.

Das Ziel des Projekts ist es zu zeigen, wie eine moderne Observability Plattform aufgebaut werden kann.

---

# Motivation

Viele Observability Plattformen wie:

* Datadog
* Grafana Cloud
* Honeycomb

bieten eine zentrale Telemetry Ingestion Layer mit:

* Authentifizierung
* Multi-Tenancy
* Telemetry Enrichment
* zentralem Routing

SpanScout demonstriert diese Architektur in vereinfachter Form.

---

# Projektziele

SpanScout soll zeigen:

* wie Telemetrie zentral ingestiert werden kann
* wie API Key basierte Telemetry Authentifizierung funktioniert
* wie eine Observability Control Plane aufgebaut wird
* wie OpenTelemetry in eine Plattformarchitektur integriert werden kann

---

# Lernziele

Das Projekt dient als praktische Übung für:

* Distributed Systems
* Observability Plattformen
* Telemetry Pipelines
* Platform Engineering
